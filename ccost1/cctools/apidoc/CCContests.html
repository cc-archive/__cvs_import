GetCurrentPollingData ()
Short Desc.: Returns the polling data for the most current contest
Description: This method returns a complex array of polling data and other contest information. Tip: dump the return value of this method using CCDebug::PrintVar() to exactly what is returned. Also note: the return values change slightly depending on whether the current user has already voted in the poll. If not, the data strunction includes the URL of the vote command as well as text to use for a link

GetOpenStatus (
array $row Reference to contest database row
)
Short Desc.: Populate a database row for a contest with specific state flags
Description: Upon returns there will be several boolean flags regarding the current state of this contest. <?

        $a['contest_taking_submissions']  // true if NOW is before deadline
        $a['contest_voting_open']         // true if voting is allowed and NOW is after deadline but before voting deadline
        $a['contest_show_results']        // true if voting is allowed and after voting deadline
        $a['contest_can_browse_entries']  // true if browing is always allowed or after contest deadline
  ?>
