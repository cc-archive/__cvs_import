class CCDatabase
include cc-database.php
Short Desc.: Wrapper class for mySQL, however only CCTable should be calling it directly.
See also: CCTable::CCTable
DBClose ()
Short Desc.: (undocumented method)
DBConnect ()
Short Desc.: Static call to ensure connection to daemon. Can be called multiple times safely.
Query (
mixed $sql single mySQL query or array of them
)
Short Desc.: Performs one (or more) mySQL queries.
QueryItem (
string $sql mySQL SELECT statement with a single column
)
Returns: string First column results from SELECT statement
Short Desc.: Retrieves a single item from a single row. Use with SELECT statment.
Description: <?
   $username = CCDatabase::QueryItem("SELECT username FROM users WHERE id = '9'\");
?>
QueryRow (
string $sql , single mySQL SELECT statement
bool $assoc = true TRUE means fetch_assoc, FALSE means fetch_row
)
Returns: array Row from database or null if results count greater or less than one.
Short Desc.: Retrieves a single row. Use with SELECT statment.
QueryRows (
string $sql mySQL SELECT statement
)
Returns: array Array with database rows inside
Short Desc.: Retrieves multiple rows. Use with SELECT statment.
Description: <?
        $rows =& CCDatabase::QueryRows("SELECT username, age FROM users WHERE age < 27\");
        foreach( $rows as $row )
        {
            // ....
        }
?>
