FileValidate (
object $formatinfo Database record of upload
)
Short Desc.: Validates a file to be of a certain type.
Description: This method is called by checking for the global '$CC_UPLOAD_VALIDATOR' and then calling $CC_UPLOAD_VALIDATOR->FileValidate($formatinfo). <?

        $format_info = new CCFileFormatInfo('/some/path/to/file');

        if( isset($CC_UPLOAD_VALIDATOR) )
        {
            if( $CC_UPLOAD_VALIDATOR->FileValidate($format_info) )
            {
                // validated ok
            }
            else
           {
               // handle problems
              $errors = $format_info->GetErrors();
           }
        }
?>
See also: CCFileFormatInfo::CCFileFormatInfo
