Short Desc.: Contest API and event callbacks
Contests (
string $contest_short_name = '', Short (internal) name of contest
string $op1 = '', Next part of url
string $op2 = '' Last part of url
)
Short Desc.: Catch all and dispatcher for many contest realted urls
Description: handles the following urls: <?
        //     contest
        //     contest/[name]
        //     contest/[name]/edit
        //     contest/[name]/submit
        //     contest/[name]/sources
        //     contest/[name]/sources/submit
        //     contest/[name]/entries
        //     contest/[name]/vote
        //     contest/[name]/results
        //     contest/[name]/delete
?>
