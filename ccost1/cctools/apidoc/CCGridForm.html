class CCGridForm extends CCForm
include cc-form.php
Derived classes: CCAdminMenuForm, CCAdminMenuGroupsForm, CCAdminRawForm, CCEditLinksForm
Short Desc.: Alternative view of HTML form controls
Description: This class sets up a form with a grid of form elements useful when editing several records at once.
