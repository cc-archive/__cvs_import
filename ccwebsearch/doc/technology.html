Requirements

We have a few technology requirements:

  • High-Level and Rapid Development: the requirements will change rapidly and we want to be able to adapt the code quickly.
  • Integration: as much as possible, code produced should be easy to integrate with other relevant code developed for/by Creative Commons.
  • Maintenance: easy to maintain given existing Creative Commons technology investments.

Data Storage

PostgreSQL. We will likely use PostgreSQL 7.4 for this development. If 7.4 turns out to be unstable in development, we will revert to 7.3.4.

Code

Python 2.3, with specific add-on modules for:

  • threading
  • RDF/XML parsing
  • database connectivity: psycopg
