Short Desc.: Wrapper for the GetID3 Library
BadPath ()
Short Desc.: Called internally when we cant find the installation of GetID3
GetFormats ()
Short Desc.: Get the default formats handled by the library
Description: Returns an array of the following structure:

<?
         $file_formats['audio-aiff-aiff'] =  array(
           'name'        => 'aif',
           'description' => 'AIFF Audio',
           'enabled'     => true,
           'mediatype'   => 'audio',
           );
?>
InitID3Obj ()
Short Desc.: Initialize the GetID3 library
Description: Sets the following parameters:

<?
            $ID3Obj = new getID3;
            $ID3Obj->option_tag_lyrics3       = false;
            $ID3Obj->option_tag_apetag        = false;
            $ID3Obj->option_tags_process      = true;
            $ID3Obj->option_tags_html         = false;

?>

Short Desc.: Called internally when GetID3 is not properly installed
